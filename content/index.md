## Markspresso

Markspresso is a LuCLI module that brews static HTML sites from Markdown content.

This documentation walks you through:

- Creating a new site
- Understanding the generated structure and configuration
- Writing content with front matter
- Designing layouts and partials
- Building navigation
- Creating posts and pages
- Running builds and watching for changes

- **Getting started** – prerequisites, installing and running your first build: [/docs/010_getting-started/](./docs/010_getting-started.md)
- **Site structure & configuration** – how `markspresso.json` controls paths and collections: [/docs/020_site-structure-and-config/](./docs/020_site-structure-and-config.md)
- **Content & front matter** – how Markspresso reads metadata: [/docs/030_content-and-front-matter/](./docs/030_content-and-front-matter.md)
