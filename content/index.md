## Markspresso

Markspresso is a LuCLI module that brews static HTML sites from Markdown content.

This documentation walks you through:

- Creating a new site
- Understanding the generated structure and configuration
- Writing content with front matter
- Designing layouts and partials
- Building navigation
- Creating posts and pages
- Running builds and watching for changes

### Where to start

- [**Getting started**](/docs/getting-started) – prerequisites, installing and running your first build
- [**Site structure & configuration**](/docs/site-structure-and-config) – how `markspresso.json` controls paths and collections
- [**Content & front matter**](/docs/content-and-front-matter) – how Markspresso reads metadata
- [**Layouts & partials**](/docs/layouts-and-oartials) – customizing HTML output
- [**Navigation**](/docs/navigation) – sidebar navigation driven by filenames: [/docs/050_navigation/](./docs/050_navigation.md)
- [**Posts & pages**](/docs/posts-and-pages) – working with collections and the `new` command
