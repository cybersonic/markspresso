Markspresso currently ships with everything needed to publish a polished static site from Markdown:

- `create` scaffolding for content, layouts, assets, and config
- `build` output with pretty URLs and collection routing
- `watch` mode for quick local iteration
- Front matter support for metadata and draft control
- Optional Lunr search and RSS/Atom feed generation

It is a pragmatic workflow for teams that want fast publishing without adding a full frontend framework build pipeline.
